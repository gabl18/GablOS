About Me

About Me

Hello, I am Gabl, a guy from Austria doing much different stuff.
I am trying to learn how to make fun Games, Art and Assets for people to enjoy. This is why I wanted to create this Website as my Portfolio to display some of my work! :D
I would say that I am a pretty creative person and have been drawing for ages. In particular Pixel Art has really grown on me ever since I first started doing it for a gamejam early 2024 - used paint.net back then wich was a huge pain haha but I have since then upgraded to aseprite and have already sunken tons of hours into it. I have published some asset-packs on itch.io (one of them even has over 1000 downloads) but also have worked on some games together with friends for gamejams while mainly doing all the art stuff. We even managed to win one of them :). Fun fact: I actually took the background image on a trip to spain.

For any comission or inquiry shoot me a Message at any of the provided Links

Vinylity

Release: 9th August 2025

Working on this game were 3 people once again, sir_lordmike, ilem and me! "Vinylity" was made in about 3.5 days for the GMTK Game Jam 2025 with the theme "LOOP" and everything was made specifically for this game. No premade assets have been used, the music (ilem), the code (sir_lordmike), the art (gabl18)- even the font have been made specifically for Vinylity.

Vinylity is a rythm game that tests your patience. The game never ends, meaning you can keep playing for how much time you want to. The songs keep LOOPing, but the game gets harder and harder.

We had such a blast working on this game, especially when implementing my own hand into the game lol. Not everything went as planned tho and we sadly couldn't get a web build running.

Tiny Globes

Release: 26th May 2025

This game was the first game in which we were 3 people: sir_lordmike, me and ilem! We only wanted to make a small game for the GameDevTVJam2025 but somehow we ended up working like 7 days or more on it lol. The theme was "Tiny World" and we came up with the game Tiny Globes, where you are a Snowglobe-Shop Owner and get requests from guests to build specific snowglobes and thus create their tiny worlds (;

ilem made an amanzing soundtrack, sir_lordmike did all the programming - and I did the art and also implemented many sounds!

We ended up being 46th out of 1014!!! Which was a amazing result. This was a really fun experience (even tho we also had a intense crunch in the end with submitting the game like 1 minute before the deadline haha)!

Lasersplinter

Release: 16th March 2025

This game was made together with sir_lordmike again, for the German game jam Puls Game Jam. The theme was "Kaputt" wich translates to destroyed in english. The Gamejam was only 48 hours long, and we messed up our Github Repo wich did cost us like 4 hours and much pain. This lead to a real crunch in the end with us almost not being able to submit the game in time (but we did it, like 3 min were remaining, phew)

We are really proud for how this game turned out - especially for the time and given it being our first real Godot game. (godot is actually a game-engine other than pygame hahaha). There were 2.5k € on the line, but we ended up getting 50th place, wich we are still very greateful for, because there were some amazing games done in this time!

The Perfect Drink

Release: 4th April 2024

THE first Game-Jam me and sir_lordmike ever participated in. We made this platformer in about a week (and yes, we almost worked like 10 hours a day on it ;_;) in python with the library "pygame" for the pygame-community-spring-jam-2024 and we ended up getting 3rd place!!!! :DD

Thinking back to this game we have sooo much that could have been done better (no coyote time, no spritesheets 💀, fixed window size, and so much more). BUT we learned sooo much from this Jam and I am really greateful that because of pygame we started participating in game jams! (but I will never go back to pygame ;_; it was hell of a pain) (;
